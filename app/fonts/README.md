# Lemon Milk Font Files

Place your Lemon Milk font files in this directory with the following names:

- `LemonMilk-Regular.otf` (or `.ttf`) - Regular weight (400)
- `LemonMilk-Medium.otf` (or `.ttf`) - Medium weight (500)
- `LemonMilk-Bold.otf` (or `.ttf`) - Bold weight (700)
- `LemonMilk-Light.otf` (or `.ttf`) - Light weight (300)

## Where to get Lemon Milk

Lemon Milk is a commercial font. You can purchase it from:
- [MyFonts](https://www.myfonts.com/fonts/marsnev/lemon-milk/)
- [Font Squirrel](https://www.fontsquirrel.com/fonts/lemon-milk)
- Other font retailers

## File Format

The font files can be in either `.otf` (OpenType) or `.ttf` (TrueType) format. The configuration will work with both.

## Alternative: Using a CDN

If you prefer to use Lemon Milk via a CDN, you can update `app/globals.css` to use `@font-face` with a CDN URL instead of local files.
